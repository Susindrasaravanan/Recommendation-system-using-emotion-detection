About

This is a project for detecting emotions and providing personalized recommendations for movies, music, games, and novels based on detected emotions.

Technologies Used

  • Emotion Detection: We use OpenCV for real-time video capture and image processing.
  • Deep Learning: We use Keras for building and training deep learning models.
  • Backend: We use Flask for the backend server implementation.
  • Frontend: We use HTML, CSS, and JavaScript for the frontend development.
  • Fonts: We use Google Fonts for typography.
  • Icons: We use Ionic Icons for iconography.
  • API: We use REST API for communication between frontend and backend.
  • Database: We use MongoDB as the database for storing data.
  • Deployment: We deploy our application on a cloud platform for scalability.
